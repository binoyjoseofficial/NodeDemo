# NodeDemo Repository

This is the NodeDemo project repository. Please start using this for all documentation and examples.

## Website

This repository has an associated website:
- Repository: https://github.com/binoyjoseofficial/NodeDemo
- GitHub Pages Settings: https://github.com/binoyjoseofficial/NodeDemo/settings/pages

## Guidelines

**Documentation**
- Add all kinds of documentation here without cluttering the main directory
- This is a public repository, so never include passwords or sensitive information

**Images and Media**
- Place images and configuration guides in dedicated folders
- For each folder containing images, add an `about.md` file describing the contents
- This structure enables automatic website generation from the repository
